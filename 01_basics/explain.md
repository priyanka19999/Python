A. What is __pycache__?
: It is a system folder for python 
:When you run Python:
   - Python converts your code into bytecode
   - It stores it in __pycache__
   - File looks like: hello_python.cpython-314.pyc
   
B. what is .pyc ?
- compiled python or Frozen binaries
- .pyc works only for imported files not for top level files. 

C. Python Inner Working ?
- python code → Byte code → Python virtual machine

D. What is PVM ?
- Python Virtual Machine 
- Used for code loop to iterate byte code .
- Also called as "Run time engine" and "Python Interpreter.

E. What is Byte code ?
- It is a Low level code and Platform Indepentent.
- It is a Python specific Interpretation
- it is not a machine code
